이용 가이드

JJU Map을 빠르게 이해하고 활용할 수 있도록 기본 사용법을 정리했습니다.

시작하기

  1. 카카오 개발자 콘솔에서 JavaScript API 키를 발급받아 index.html과 map.js에 적용합니다.
  2. 각 카테고리 페이지에 들어갈 시설 데이터를 JSON 또는 JS 모듈로 정리합니다.
  3. map.js에 마커 생성, 인포윈도우, 카테고리 필터 로직을 구현합니다.
  4. 오라클 클라우드 서버에서 Nginx 문서 루트를 프로젝트 폴더로 연결하고 배포합니다.

페이지 구조

모든 페이지는 공통 헤더/푸터를 사용하며, 지도 영역과 시설 리스트 영역으로 구성됩니다. CSS는 style.css에서 관리합니다.
