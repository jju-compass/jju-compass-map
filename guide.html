📘 이용 가이드

전주대 주변 탐색지도 사용 방법을 안내합니다

🏠 메인 소개 이용가이드

사용 방법

1. 검색 바 사용하기

메인 페이지 상단의 검색 바에 원하는 장소나 카테고리를 입력하세요.
예: "한식", "카페", "편의점" 등을 검색하면 해당 카테고리 페이지로 이동합니다.

2. 카테고리 선택하기

메인 페이지에서 원하는 카테고리 카드를 클릭하세요.
음식점, 카페, 편의점, 약국, 병원, 은행/ATM, 문구점, 미용실, PC방, 헬스장, 노래방 등을 선택할 수 있습니다.

3. 지도에서 장소 확인하기

카테고리 페이지에 들어가면 왼쪽에 장소 목록, 오른쪽에 지도가 표시됩니다.
마커를 클릭하거나 목록에서 장소를 선택하면 상세 정보를 확인할 수 있습니다.

4. 도보 경로 확인하기

"내 위치 시작" 또는 "시작 지점 지정" 버튼을 클릭하여 출발지를 설정하세요.
이후 목적지를 클릭하면 경로가 애니메이션으로 표시됩니다.

5. 모바일에서 사용하기

모바일 브라우저에서도 동일하게 사용 가능합니다.
화면이 자동으로 세로 배치로 최적화되어 표시됩니다.
