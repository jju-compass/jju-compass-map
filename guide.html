📘 이용 가이드

전주대 주변 탐색지도 사용 방법을 안내합니다

🏠 메인 소개 이용가이드 지도 보기

사용 방법

1. 카테고리 선택하기

메인 페이지에서 원하는 카테고리 카드를 클릭하세요.
음식점, 카페, 편의점, 약국, 병원, 은행/ATM, 문구점, 미용실, PC방, 헬스장, 노래방 등을 선택할 수 있습니다.

2. 검색 바 사용하기

지도 페이지 상단의 검색 바에 원하는 장소나 키워드를 입력하세요.
예: "한식", "카페", "편의점" 등을 검색하면 해당 장소들이 지도에 표시됩니다.

3. 지도에서 장소 확인하기

왼쪽 사이드바에서 카테고리를 선택하면 지도에 마커가 표시됩니다.
마커를 클릭하거나 오른쪽 목록에서 장소를 선택하면 상세 정보를 확인할 수 있습니다.

4. 도보 경로 확인하기

화면 왼쪽 하단의 "경로 보기" 버튼을 클릭하세요.
"내 위치 사용" 또는 "지도에서 선택"으로 출발지를 설정한 후,
목적지 마커를 클릭하면 도보 경로가 애니메이션으로 표시됩니다.

5. 모바일에서 사용하기

모바일 브라우저에서도 동일하게 사용 가능합니다.
화면이 자동으로 세로 배치로 최적화되어 표시됩니다.

6. 다크 모드

시스템 설정에 따라 자동으로 다크 모드가 적용됩니다.
야간에 눈의 피로를 줄이고 배터리를 절약할 수 있습니다.

💡 유용한 팁

  • 빠른 이동: 목록에서 장소를 클릭하면 지도가 해당 위치로 자동 이동합니다.
  • 상세 정보: 마커를 클릭하면 전화번호, 주소, 카카오맵 상세보기 링크를 확인할 수 있습니다.
  • 경로 초기화: "경로 지우기" 버튼으로 표시된 경로를 제거할 수 있습니다.
  • 키보드 사용: Tab 키로 버튼과 링크를 이동하고 Enter로 선택할 수 있습니다.
