JJU Map 소개

JJU Map은 전주대학교 학생과 교직원이 주변 시설 정보를 한눈에 확인하도록 돕는 맞춤형 지도 서비스입니다.

우리가 해결하려는 문제

  • 처음 전주대를 찾는 학생이 학교 주변 지리를 빠르게 파악하기 어렵습니다.
  • 약국·편의점처럼 급히 찾아야 하는 시설을 기존 지도앱에서 찾기까지 시간이 걸립니다.
  • 학교 특화 정보(학생 선호 장소, 인기 카테고리 등)가 부족합니다.

서비스 핵심 기능

카테고리별 지도

음식점, 카페, 약국, 편의점 등 주요 시설을 카카오맵 기반으로 빠르게 검색합니다.

맞춤형 UI

모바일과 PC 환경 모두에서 사용할 수 있는 반응형 인터페이스를 제공합니다.

확장 가능한 구조

리뷰, 즐겨찾기, 검색 필터 등 추가 기능을 쉽게 붙일 수 있도록 페이지를 모듈화합니다.

개발 로드맵

  1. 기본 지도, 카테고리 페이지, 공통 UI 구성요소 구현
  2. 시설 데이터 수집 및 마커/리스트 표시 로직 완성
  3. 사용자 리뷰, 북마크, 공유 기능 추가
  4. 오라클 클라우드 서버에 배포하여 실서비스 운영
