프로젝트 소개

전주대 주변 탐색지도는 전주대학교 학생들을 위한 지도 서비스입니다.

카카오맵 API를 활용하여 주변 음식점, 카페, 편의시설 등을 쉽고 빠르게 찾을 수 있습니다.

카카오맵 기반

정확한 위치 정보와 최신 데이터를 제공합니다

다양한 카테고리

음식점부터 편의시설까지 15개 이상의 카테고리

반응형 디자인

PC와 모바일 모두 최적화된 사용자 경험

기술 스택

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • API: Kakao Maps JavaScript SDK v2
  • Deployment: Oracle Cloud, Nginx, Let's Encrypt SSL
  • Repository: GitHub
