빠른 시작 가이드

  • 카카오맵 API 등록 카카오 개발자 콘솔에서 JavaScript 키를 발급받은 뒤 `map.js`에 추가하세요.
  • 시설 데이터 입력 `data/` 디렉터리를 만들어 카테고리별 JSON을 정의하고, 지도와 리스트에 반영할 수 있습니다.
  • 오라클 클라우드 배포 GitHub에서 최신 코드를 `git pull`한 뒤 Nginx 문서 루트와 연결하면 즉시 서비스가 구동됩니다.
